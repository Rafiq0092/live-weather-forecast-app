# 🌦️ Weather Forecast Web Application

A responsive web application that provides **real-time weather information and forecasts** for any city using a weather API.
Users can search for locations and view current weather conditions such as **temperature, humidity, wind speed, and weather description**.

---

## 📌 Features

* 🌍 Search weather by **city name**
* 🌡️ Displays **current temperature**
* 💧 Shows **humidity levels**
* 🌬️ Wind speed information
* ☁️ Weather condition description
* 📱 Responsive design (mobile-friendly)
* ⚡ Fast API-based weather data retrieval

```bash

```

```bash

```

3. Open `index.html` in your browser

## 🔑 API Setup

This project uses a weather API such as **OpenWeatherMap**.

Steps:

1. Go to
   https://openweathermap.org/api

2. Create a free account

3. Generate an **API Key**

4. Replace the API key inside `script.js`

```javascript
const apiKey = "YOUR_API_KEY";
```
